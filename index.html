    Zen Templates

    Simple, elegant HTML5 templates.

    Zen Templates is a web template language for generating clean, standards-compliant web sites from templates that will render on their own in a standard web browser. All template files are themselves fully valid HTML5 that can be previewed by a designer without needing to run the full application.

    The key goals of Zen Templates are:

    • Seamless HTML 5 integration - templates and output should be fully compliant, renderable HTML 5 markup.
    • Semantic parsing - semantic information offered in classes and IDs should be usable in place of custom tags or attributes.
    • Intuitive rendering - if an element is backed by a collection in the model, repeat the element for each item in the collection.
    • Platform-agnostic - the same templates should be renderable by applications running on Java, .NET, PHP, Python, Ruby, node.js, and so on.

    Platform implementations in progress:

    • Java (original prototype - official implementation pending)
    • C#: http://github.com/aprice/ZenTemplatesDotNet/
